### Rodando localmente


<h4>1º | Faça o clone do repo: </h4>

<code>
    git clone https://github.com/future4code/paiva-marketplace2.git
</code>

<br>
<br>

<h4>2º | Instale as dependências: </h4>

<code>
   npm install
</code>

<br>
<br>

<h4>3º | Inicie o servidor: </h4>

<code>
   npm run start
</code>
